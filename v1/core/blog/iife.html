Immediately-Invoked Function Expression is JavaScript design pattern to limit pollution of the global name space and also immediately call a function.

Meaning that instead of declaring function you are only ever going to use once you can use IIFE to declare a function then immediately run it leaving the name of the function free and all variables inside of the function out of the global name space.

Standard way

var someFunc = function() {
  console.log("Done something");
};
someFunc();

Immediately-Invoked Function Expression

(function() {
  console.log("Done something");
})();

When to use

When a function is to be used Immediately and you don't want the variables or functions inside to pollute the global name space.
