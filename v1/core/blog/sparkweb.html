SparkWeb is an IM client most commonly used in combination with OpenFire. I will show you how to install it on Ubuntu 14.04.

Step 1:

Make sure you system is updated.

sudo apt-get update && sudo apt-get upgrade


Step 2:

Download dependencies.

sudo apt-get install lamp-server^


Step 3:

Navigate to the correct directory.

cd /var/www/html


Step 4:

Create directory for SparkWeb.

sudo mkdir -p webchat && cd


Step 5:

Download SparkWeb.

wget http://www.igniterealtime.org/downloadServlet?filename=sparkweb/sparkweb_0_9_0.zip


Step 6:

Unpack .zip file.

sudo unzip -j sparkweb_0_9_0.zip


Step 7:

Remove .zip file.

sudo rm sparkweb_0_9_0.zip


Step 8:

Rename SparkWeb to index.

sudo mv SparkWeb.html index.html


Step 9:

Now change edit index.html to work with your local OpenFire server.

sudo nano index.html

Change only line 32

from 'igniterealtime.org' to 'server.local'


Step 10:

Open your browser and go to your http://yourdomain.com/webchat
