OpenFire is an IM and group chat server that uses Apache. In this post I will you teach you how to install OpenFire on Ubuntu Server 14.04. Lets get started.

Step 1:

Make sure your sever is updated.

sudo apt-get update && sudo apt-get upgrade


Step 2:

Install openjdk 7 which is required for OpenFire.

sudo apt-get install openjdk-7-jre


Step 3:

Download OpenFire onto the server.

wget http://www.igniterealtime.org/downloadServlet?filename=openfire/openfire_3_9_3.tar.gz


Step 4:

Rename file.

mv downloadServlet?filename=openfire%2Fopenfire_3_9_3.tar.gz openfire_3_9_3.tar.gz


Step 5:

Unpack the .tar.gz file.

tar -xvxf openfire_3_9_3.tar.gz


Step 6:

Move OpenFire to /opt

sudo mv openfire /opt


Step 7:

Edit config file.

sudo nano /opt/openfire/conf/openfire.xml

Uncomment the tag and put your servers public IP in between the tags.


Example:

<network>192.168.7.20</network>


Step 8:

Now start you server.

/opt/openfire/bin/openfire start


Step 9:

Go through setup in browser.

Choose you language and click continue. Then choose embedded database and you will be confronted with admin configuration panel. Set up the account with your information. Then login using username: admin and what ever password you set the password as.


Step 10:

Set up your users and start talking!
