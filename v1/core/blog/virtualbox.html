Ubuntu is a great OS and lots of fun but before you go wiping your hard drive and installing Ubuntu maybe you should try it out on your current system first.

Virtual Box is a virtual machine program for Windows, Linux and OSX. It's great for running other operating systems in your current computer and is what were going to be using today.

To install Ubuntu for your system simply go to Ubuntu's website and download the latest 32 bit ISO for desktop. Then go to the Virtual Box website and download the client for windows. Next install Virtual Box using the recommended settings and click "New", point it to the ISO you just downloaded. Lastly run through the install setup for Ubuntu and enjoy.
