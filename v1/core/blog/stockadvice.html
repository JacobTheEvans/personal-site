I recently got the stock trading fever. Now I could hire a broker and have them manage my stocks or I could use the magic of python to help me out.

So I made this program to assist in managing stocks. The basic idea of this project is that instead of checking the stock market yourself you let python do it and report back. So I used the urllib2 library to check google finance for all current holdings in the database(provided by the user). When gross profit for a certian stock would be more then 3.5% if sold It informs the user.
