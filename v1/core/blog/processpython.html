Pexpect is a python module for running child processes inside of linux. I recently ran into a problem with running programs in linux while still maintaining control of the procces itself while in python.

The pexpect is a module for controlling terminal commands in python for programs like ssh, ftp, telnet but it can be used for so much more. Using pexpect.run() method you can run programs from terminal such as a browser. For example I can run other python programs from pexpect by using a simple line of code like below.

  pexpect.run("python /path/example.py")

Although it is a bit cheaty it is an awesome way to make a master control program in python.
